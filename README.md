# Bridge Inspection App

这是一个用于桥梁检查的应用程序，它允许用户添加和查看桥梁信息。

## 功能

- 添加新的桥梁信息
- 查看已有的桥梁信息
- 对桥梁信息进行搜索和过滤
- 对桥梁信息进行编辑和删除

## 安装

1. 克隆此仓库到本地。
2. 在 Visual Studio 中打开项目。
3. 确保已安装所有必要的 NuGet 包。

## 使用

运行应用程序，然后按照界面提示进行操作。

## 测试

1. 在 Visual Studio 中打开项目。
2. 打开测试资源管理器。
3. 运行所有测试。
